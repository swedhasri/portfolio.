# Swedha Sri S - Portfolio

This is the personal portfolio website for Swedha Sri S, a Computer and Communication Engineering student and Full Stack Developer.

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Dark/Light Mode**: Toggleable theme.
- **Animations**: Typing text, glassmorphism effects, and scroll animations.
- **Tech Stack**: HTML, CSS, JavaScript (No frameworks).

## Setup
To run this project locally:

1. Clone the repository.
2. Open the `index.html` file in your browser.
   OR
3. Run a local server (e.g., using VS Code Live Server or Python/Node).

```bash
# Using Python
python -m http.server 8000

# Using Node (if server.js is present)
node server.js
```
